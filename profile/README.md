# Intelligent Class Ecosystem
# 冰核智慧课堂生态系统(Intelligent Class Ecosystem)：更智能、更便捷、更全面的课程管理系统
</div>

**冰核智能教务系统** 旨在为课堂课程表管理提供智能化支持，创造性地对传统电子课表软件的课表管理逻辑进行改革，同时加入更多实用班级多媒体管理工具，为教学人员提供更丰富、更智能、更友好的教学管理服务。
简要介绍
该系统分为 **教室终端**(显示课程/提示和其它工具)和**教务管理系统**(Web版及多端适配)。教务管理系统可以实现以下场景：
- 教师申请调课
- 教务科监控设备和修改课表
- 班主任控制本班设备
- 年段长查看全年段设备状态
- ......   

系统通过运作/既定时间线分离机制，兼顾课表稳定性与临时调整需求，实现教学流程的动态管理。   

## 我们的愿景
随着ClassIsland和ClassWidgets等电子课表软件在各所学校不断被推广和使用，我们意识到，需要一个拥有以下功能的系统和软件服务于各个学校：
- 方便灵活的课程管理
- 明确的分权账号系统
- 更加美观易用的界面
- ......
而这个系统是自上而下的：从教务科到年段长班主任，再到科任老师和班级，我们希望做到师生共用，校园同享。
