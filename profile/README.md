# Intelligent Class Ecosystem
# 冰核智慧课堂生态系统(Intelligent Class Ecosystem)：更智能、更便捷、更全面的课程管理系统
</div>

**冰核智能教务系统** 旨在为学校教务与班级管理提供智能化支持，创造性地对传统课表工具进行改革，同时加入更多实用班级管理工具，为老师提供更丰富、更智能、更友好的教学管理服务。
## 项目介绍
该系统分为**教室端**和**教师端**。
**教室端**可以实现以下场景：
- 呈现最新教务信息
- 课表调整的呈现
- 相关课程事务的同步
- 班级白板的集控管理
- ......

**教师端**可以实现以下场景：
- 教务需求的提交与审批（如：调课、代课等）
- 日程安排的修改与通知
- 向教室端发送相关课程事务（如：消息、作业、备忘事项）
- 班级设备状态的统一监控与运维
- ......   

## 我们的愿景
随着ClassIsland和ClassWidgets等学生自行编写的电子课表软件和电教工具在各校不断被推广和使用，
在日益发展下的电教设备和相关需求下，
我们迫切需要一个拥有更完整功能的系统和软件服务于各个学校：
- 更加精简易读的界面
- 方便灵活的校园事务与同步
- 与职务绑定的明确分权账号系统
- ......

所以这个系统应该是自上而下的：从学校高层到年级组长到班主任，再到科任老师和班级，我们希望做到师生共用，校园同享，省去不必要的麻烦，促进构建学校完整的教务体系。
